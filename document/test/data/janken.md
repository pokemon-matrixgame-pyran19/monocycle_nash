# じゃんけんゲーム
じゃんけんモチーフのゲームは均衡解などが手で解ける。正しい解になるか確認。
利得行列の行列要素は√3の倍数にしないと相性ベクトルがややこしくなる注意


A=2√3×(
0 1 -1
-1 0 1
1  -1 0
)

等価なキャラクターは次
p1,v1 = 0, 2, 0
p2,v2 = 0, -1,  √3
p3,v3 = 0, -1, -√3

次も可
p1,v1 = -2     ,    3,  1+√3
p2,v2 =  1 +√3,    0,  1+√3
p3,v3 =  1 -√3,    0,  1-√3

均衡解は1/3,1/3,1/3

均衡解と純粋戦略を対戦させた利得が乖離度
利得行列を均衡解に作用させると乖離度の一覧が得られる。

乖離度: (0,0,0)


構築ゲームの純粋戦略は12,23,13
構築ゲームの利得行列B

B= 2√3×(
0 1 -1
-1 0 1
1  -1 0
)


構築ゲームは純粋戦略の並べ方で行列の順番かわる。12,13,23の順の場合次
B= 2√3×(
0 -1 1
1 0 -1
-1  1 0
)

構築ゲームの純粋戦略の並べ方は固定するので、実際に使う方の並べ方でのみテスト
