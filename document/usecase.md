# プログラムのユースケース



1. 利得行列Aを与えてそれに対する均衡戦略を出力
2. 利得行列Aを与えてそれに対する各純粋戦略の適合度を出力
3. 利得行列Aに対して特定の作り方をした利得行列と比較(近似の妥当性チェック)
4. チームごとにパワー、相性を入力して、対応する適合度出力
    - チームごとにパワー、相性を入力して、対応する利得行列出力。
5. 単体の利得行列Aを入力して構築ゲームの分析
    - 構築ゲームの利得行列出力
    - 適合度出力
6. チームのパワー、相性をランダム生成して、それらに対して解析。(近似の妥当性)
